Unity Distribution Portal (UDP) is now in public preview.#

To get started, download and install either of the two packages:

  • UDP Package for Unity IAP
    If you’re already using Unity IAP in your game, we recommend you use the Unity IAP Package.

  • Standalone SDK
    If you are not currently using Unity IAP, we recommend you use the stand-alone UDP SDK. Its implementation is similar to that of mainstream IAP SDKs, and slightly more direct.

    c29b0ecc59426c732917e124e44259605b02ccdf

Get the UDP package

  • from Unity’s Package Manager, or
  • from the Asset Store

If you use Unity IAP, import Unity IAP 1.22 (or above) which readily includes UDP.

Find more information in the UDP documentation.
