Implementing Unity IAP#

If you want to use Unity IAP instead of UDP IAP, set up Unity IAP according to Setting Up Unity IAP.

Note: If you choose to implement Unity IAP instead of UDP IAP, you just need to implement Unity IAP APIs.

When the Unity IAP is enabled, take the following steps to set up the Unity IAP for UDP.

  1. Select UDP as the target platform for the game to build.

  1. To configure the IAP cataolog, select IAP Catalog.

  2. Make sure that Unity IAP and UDP IAP have the same configuration.

You can find IAP Catalog under both Unity IAP and Unity UDP. Make sure these IAP catalogs have the same settings in the following fields:

  • Name, which specifies the name of the item
  • Product ID, which specifies the unique identifier of the item
  • Type, either consumable or non-consumable
  • Price, which specifies the price of the item
  • Description, which describes the item
