Overview of Unity Distribution Platform#

When you develop your game with the Unity Editor, you can use Unity Distribution Platform (UDP) to manage your game and in-app purchases (IAP). UDP is a game distribution platform that enables you to configure your games, manage IAP, and publish your games to Unity’s partner stores collectively. Unity’s partner stores includes Google, Amazon, Samsung, CloudMoolah, and Xiaomi.

Publishing your game to partner stores is a 4-step process:

  • Configure and build your game for UDP
  • Implement IAP
  • Manage your game on UDP portal
  • Publish your game to partner stores

Prerequisites

  • UDP is supported in Unity 5.6.1 or higher.
  • Currently, UDP only supports android.

Also See

  • For FAQs and more information on UDP, visit the FAQ.
  • For comprehensive information on all the topics covered in this document, visit Unity IAP UDP Integration Guide.

Configuring and building your games for UDP#

Building your games for UDP requires the following steps:

  1. Import UDP SDK
    If you are currently using Unity IAP, enable Unity IAP and install Unity SDK Package (with Unity IAP). If you are not currently using Unity IAP, install Stand-alone UDP SDK.
  2. Generate a Unity client and test accounts
    In the Unity Editor, select Window > Unity UDP > Game Settings.
    Select the Game Settings to view its Inspector, then select Generate Unity Client to populate credentials UDP will need for client-to-server communications.
    In the Inspector of the Game Settings, generate test accounts by providing emails and passwords.
  3. Build your games
    Create a local Android APK build (in the Editor, File > Build Settings… > Android > Build; see documentation on building for Android).

For detailed information, please refer to Configuring and building your game for UDP.

Implementing IAP#

You can choose either UDP IAP or Unity IAP to implement in-app purchases in your app.
If you choose UDP IAP, configure the IAP catalog through the IAP GUI (Window > Unity UDP > IAP Catalog).

If you choose Unity IAP, configure the IAP catalogs of Unity UDP (Window > Unity UDP > IAP Catalog) and Unity IAP (Window > Unity IAP > IAP Catalog). Make sure the two IAP catalogs have the same setting.

You can also manage IAP items on UDP portal.

For detailed information, please refer to Implementing IAP.

Managing your game on UDP portal#

With UDP portal, you can:

  • Create and edit your profile
  • Create and edit your games
  • Push your game file to UDP portal
  • Create in-app purchasable items

For detailed information, please refer to Managing your game on UDP portal.

Publishing your game to partner stores#

Before you push your game to partner stores, make sure you have taken the following steps:

  • Maintain your profile.
  • Add your stores.

On the UDP portal,navigate to your project, then choose PUBLISH GAME. Identify your game, select the stores you wish to publish to, and choose Publish.

For detailed information, please refer to Publishing your game to partner stores.
